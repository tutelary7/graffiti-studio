📷 카메라 시점 (자동 합성)

각 캐릭터마다 카메라 기준 상대 각도를 계산해서 8면 시트에서 알맞은 그림을 자동 선택. 거리에 따라 크기 + 앞뒤 순서 자동 처리.

장면 데이터 (JSON)

메타데이터만 저장됨 (스프라이트 이미지는 별도). 나중에 기존 프로젝트의 캐릭터 라이브러리와 연결 시 ID 매핑용으로 사용.
